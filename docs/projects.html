Vibe Web Dev

I built this website to explore and learn about AI tools like ChatGPT and GitHub Copilot. With minimal prior experience in web development, I relied heavily on AI assistance—about 80% of the code, including all the CSS, most of the HTML, and many of the images, was generated with the help of these tools. This project shows how AI can empower developers to create complex websites quickly and effectively.

GameBoy Repairman

I restore broken Game Boys purchased from eBay, a hobby that has developed my technical abilities. Through these projects I’ve gained experience in safely removing electronic components, soldering and installing replacements, and repairing or upgrading screens for improved performance. Much of my learning has been guided by Odd Tinkering’s excellent repair tutorials on YouTube, which have been invaluable in building my repair expertise.
